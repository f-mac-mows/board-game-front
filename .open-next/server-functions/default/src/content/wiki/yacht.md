# 🎲 야추 다이스 (Yacht Dice) 완전 정복 가이드

야추 다이스는 5개의 주사위를 사용하여 정해진 조합(족보)을 완성하고, 최종 점수가 상대보다 높으면 승리하는 전략적 주사위 게임입니다. 운과 판단력이 동시에 필요한 게임의 규칙을 상세히 알아봅시다.

---

### 1. 게임 진행 방식

1. **주사위 굴리기**: 자신의 차례가 오면 5개의 주사위를 동시에 던집니다.
2. **고정(Keep) 및 다시 굴리기**: 
   - 나온 주사위 중 마음에 드는 것은 그대로 두고(**고정**), 나머지 주사위만 선택해서 다시 굴릴 수 있습니다.
   - 다시 굴리기는 한 차례에 **최대 2번**까지 가능합니다. (첫 투구 포함 총 3번의 기회)
3. **점수 기록**: 3번의 기회가 끝나면(또는 중간에 만족하면) 반드시 점수판의 빈칸 중 하나를 선택해 점수를 기록해야 합니다.
4. **게임 종료**: 모든 플레이어가 점수판의 모든 칸(12칸)을 채우면 게임이 종료되며, **최종 합계 점수가 높은 사람이 승리**합니다.

---

### 2. 점수 족보 (Categories)

#### 📋 상단 항목 (Upper Section)
상단 항목은 해당 숫자가 나온 주사위 눈의 합을 기록합니다.

| 항목 | 설명 | 점수 예시 |
| :--- | :--- | :--- |
| **Ones** | 1이 나온 주사위의 합 | 1, 1, 1, 2, 5 → **3점** |
| **Twos** | 2가 나온 주사위의 합 | 2, 2, 4, 4, 6 → **4점** |
| **Threes** | 3이 나온 주사위의 합 | 3, 3, 3, 3, 1 → **12점** |
| **Fours** | 4가 나온 주사위의 합 | 4, 4, 1, 2, 6 → **8점** |
| **Fives** | 5가 나온 주사위의 합 | 5, 5, 5, 2, 3 → **15점** |
| **Sixes** | 6이 나온 주사위의 합 | 6, 6, 1, 4, 5 → **12점** |

> **✨ 상단 보너스 (35점)** > 상단 6개 항목의 합계가 **63점 이상**일 경우, 보너스 점수 **+35점**을 추가로 획득합니다.  
> (팁: 모든 항목에서 주사위 3개씩만 기록해도 딱 63점이 됩니다!)

#### 📋 하단 항목 (Lower Section)
특정 조합을 완성해야 하며, 조건에 맞지 않는 칸에 기록할 경우 **0점** 처리됩니다.

| 항목 | 설명 | 점수 |
| :--- | :--- | :--- |
| **Choice** | 주사위 5개 눈의 총합 | 주사위 눈의 전체 합 |
| **4 of a Kind** | 같은 눈이 4개 이상일 때 | 주사위 눈의 전체 합 |
| **Full House** | 같은 눈 3개 + 같은 눈 2개 | **고정 25점** |
| **S. Straight** | 이어지는 숫자 4개 (예: 1-2-3-4) | **고정 15점** |
| **L. Straight** | 이어지는 숫자 5개 (예: 1-2-3-4-5) | **고정 30점** |
| **Yacht** | 같은 눈 5개 (예: 6-6-6-6-6) | **고정 50점** |

> **💡 Yacht 활용 팁** > **Yacht(야추)** 는 가장 강력한 족보입니다. 만약 야추를 달성했는데 이미 야추 칸이 차 있다면, **Full House** 칸에 기록해도 조건을 만족하므로 25점을 챙길 수 있습니다.

---

### 3. 수학적 기대값과 전략

상단 보너스를 받기 위한 목표 점수를 $S$라고 할 때, 우리가 달성해야 하는 조건은 다음과 같습니다.

$$
Bonus = 
\begin{cases} 
35 & \text{if } \sum_{i=1}^{6} (i \times n_i) \geq 63 \\
0 & \text{otherwise}
\end{cases}
$$

여기서 $n_i$는 숫자 $i$가 기록된 주사위의 개수입니다. 

1. **상단 보너스 우선**: 하단 족보가 화려해도 보너스 35점을 놓치면 이기기 어렵습니다. 4, 5, 6번 칸에서 최대한 많은 개수를 확보하세요.
2. **0점 전략**: 도저히 조합이 안 나올 때는 가장 확률이 낮은 `Yacht`나 점수가 낮은 `Ones` 칸을 과감히 0점으로 채워 다른 기회를 노려야 합니다.
3. **유연한 판단**: 3번째 주사위를 굴리기 전, 현재 남은 칸 중 어떤 칸을 채우는 것이 가장 기대 점수가 높을지 판단하는 것이 핵심입니다.

---

### 4. 승리 조건 요약
- 모든 플레이어가 12번의 턴을 마쳐 점수판을 모두 채워야 합니다.
- **최종 점수 = 상단 합계 + 보너스(상단 합계 63점 이상 시) + 하단 합계**
- 합산 점수가 상대방보다 1점이라도 높으면 당신의 **승리**입니다!
